= FreeBSD 特性

== FreeBSD 提供许多独特的功能

无论应用场景如何，操作系统都应充分利用所有可用资源。

FreeBSD 在性能、网络和存储方面的专注，结合易于管理和全面的文档，充分发挥了计算机的潜力。

== 基于 4.4BSD 的完整操作系统

link:https://freebsdfoundation.org/freebsd/timeline/[FreeBSD 的根源] 是加州大学伯克利分校计算机系统研究组的 *BSD* 软件发行版。

数十年的发展造就了先进的可扩展性、网络性能、管理工具、文件系统支持、安全性等特性。

FreeBSD 遍布互联网，应用于核心路由器产品、运行根域名服务器、托管大型网站，并作为广泛使用的桌面操作系统的基础。

[[openzfs]]
=== OpenZFS

ZFS 不仅仅是一个文件系统，它与传统文件系统有本质不同。

将传统上独立的软件 RAID、卷管理器和文件系统角色结合起来，赋予了 ZFS 独特的优势。

ZFS 的三个主要设计目标：

* 数据完整性
* 池化存储
* 性能

[[zfs-boot-environments]]
=== ZFS 引导环境

ZFS 引导环境是系统特定预选部分的引导克隆/快照。

使用场景包括：

* 对系统进行防弹式升级/更改
* 在升级或更改系统之前创建安全的回退 ZFS 引导环境
* 更新新的（非活动）环境而不改变活动环境
* 在 jail 中执行升级并测试结果
* 将 ZFS 引导环境复制/移动到另一台机器
* 重大重新配置（Bareos/Postfix/...）
* 用一个配置好的引导环境大规模填充大量服务器
* 裸机备份解决方案
