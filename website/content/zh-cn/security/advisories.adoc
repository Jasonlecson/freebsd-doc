= FreeBSD 安全公告

本页面列出 FreeBSD 项目的官方安全公告 (Security Advisories)。

强烈建议所有 FreeBSD 用户关注这些公告，并及时应用相关补丁或升级。

== 如何获取最新安全公告

* 订阅 link:../community/mailinglists/[freebsd-security-notifications] 邮件列表
* 定期查看本页面
* 使用 `freebsd-update` 工具自动获取安全更新
